Per-watch lookout

Watch lookout code: 
{{watch.get('uid')}}
To checkin for {{watch.get('name')}}, hit this URL: 
curl {{$parent.data.host}}/lookout/{{watch.get("uid")}}
To include a message, add a query parameter: 
curl {{$parent.data.host}}/lookout/{{watch.get("uid")}}?m=Got%20it
To set status, add an "s" or "status" query parameter: 
curl {{$parent.data.host}}/lookout/{{watch.get("uid")}}?s=success
Hit the lookout after your cron job is completed: 
doSomeWork && curl {{$parent.data.host}}/lookout/{{watch.get("uid")}}?s=success
